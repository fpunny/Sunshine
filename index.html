Introduction to positioning

What is positioning?

Positioning is an CSS properity that defines how an element is positioned relative to other elements on a page. (In case you dont know, element is the equivalent of saying thing for web)

There are 5 types of positioning:
  • static
  • relative
  • fixed
  • absolute
  • sticky
*We only care about the highlighted ones

What is relative & absolute positioning?
Relative

Positioning the element relative to where it should be (if static). In layman terms, it shouldn't affect how it is positioned, but now it can move around if given properities.

Basic Element Stacking

Code
  • <div class="grey lighten-3" style="max-width: 300px; height: 300px; border: 1px solid rgba(0,0,0,0.2); position: relative;">
    • <div class="blue lighten-3" style="width: 60%; height: 60%; border: 1px solid rgba(0,0,0,0.2); position: absolute; opacity: 0.4;"></div>
  • </div>
Notes

Blue: The "container" of the absolute element has to be relative, or else the absolute element wouldn't be relative to the container but the body.

Green: Cool fact, percentage would also be relative to the first positioned parent. (else body)

Red: Has to be absolute to move around inside container, it would "take up no space", meaning other non-positioned elements wouldn't be affected by this absolute element.

2+ Element Stacking

Code
  • <div class="grey lighten-3" style="max-width: 300px; height: 300px; border: 1px solid rgba(0,0,0,0.2); position: relative;">
    • <div class="blue lighten-3" style="width: 60%; height: 60%; border: 1px solid rgba(0,0,0,0.2); position: absolute; opacity: 0.4;">
      • <div class="red lighten-3" style="width: 30%; height: 30%; border: 1px solid rgba(0,0,0,0.2); position: absolute; top: 20%; left: 50%; opacity: 0.6;"></div>
    • </div>
    • <div class="green lighten-3" style="width: 60%; height: 60%; border: 1px solid rgba(0,0,0,0.2); position: absolute; top: 20%; left: 50%; opacity: 0.4;"></div>
  • </div>
Notes

Blue: We can have 2 absolutes in a relative and it's fine, plus they don't affect each other.

Green: Absolute in an absolute works too! Because absolute is relative positioning to the first POSITIONED parent, and absolute is a position.

Red: This is something like pushing the element, there are 4 orientations - top, bottom, left, right. Note that the the percent means that there is a space of about 20%/50% of the parent's width/height from the "insert orientation" edge of the parent. Also note they cancel each other out (etc. left: 20px and right: 10px = left: 10px)

Vertical Centering with Positioning

Code
  • <div class="grey lighten-3" style="max-width: 300px; height: 300px; border: 1px solid rgba(0,0,0,0.2); position: relative;">
    • <div class="blue lighten-3" style="width: 60%; height: 60%; border: 1px solid rgba(0,0,0,0.2); position: absolute; opacity: 0.4; top: 0; bottom: 0; margin: auto 0;"></div>
  • </div>
Notes

Blue: MUST add if you're doing vertical centering for compatibility reasons from experience.

Green: This is what actually centers the thing (more specifically the auto part of margin), tbh I have no idea why/how, it just is. Note if you don't know the syntax, first number is for top/bottom and second is for left/right.

Complete Centering with Positioning

Code
  • <div class="grey lighten-3" style="max-width: 300px; height: 300px; border: 1px solid rgba(0,0,0,0.2); position: relative;">
    • <div class="blue lighten-3" style="width: 60%; height: 60%; border: 1px solid rgba(0,0,0,0.2); position: absolute; opacity: 0.4; right: 0; left: 0; top: 0; bottom: 0; margin: auto;"></div>
  • </div>
Notes

Blue: Yup you guessed it. Complete centering is just vertical and horizontal stuff all together.

Green: Both stuff combined, note that the sytnax for if there is 1 thing means its for all (top, bottom, left, right).
